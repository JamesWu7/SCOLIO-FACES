# SCOLIO-FACES

Facial phenotype-assisted hierarchical screening and etiology stratification for scoliosis research.

- Public interactive app: https://huggingface.co/spaces/wuxinjie/SCOLIO-FACES
- Source code: https://github.com/JamesWu7/SCOLIO-FACES
- Model weights: stored in the private Hugging Face model repository `wuxinjie/SCOLIO-FACES-models`; model files are not included in this GitHub repository.

The app is for research demonstration and clinical triage support only. Model outputs are relative support scores and are not standalone diagnostic conclusions.
